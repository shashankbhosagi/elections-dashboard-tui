create a python venv
```bash
python -m venv dash_env
```

install the requirements
```bash
pip install -r requirements.txt
```

enjoy the TUI Dashboard for elections
```bash
python elections_dashboard.py
```

This TUI fetches the html source and scrapes it for data and shows the results in cool TUI. THis is for people who want to keep an eye on elections results but also dont want to waste resources by opening a tab in thier browser.

This script fetches the html page every 3 minutes and you can press `r` to refresh immediately so no worries of IP ban lol.
